# 二维码生成器部署指南

## 🌐 让手机可以访问的3种方法

### 方法1️⃣：部署到Vercel（永久免费）

1. 安装Vercel CLI
```bash
npm install -g vercel
```

2. 登录Vercel
```bash
vercel login
```

3. 部署项目
```bash
vercel --prod
```

4. 获得公网地址，例如：`https://your-app.vercel.app`

---

### 方法2️⃣：手机开热点给电脑连

这是最快的临时方案：

1. 手机开启个人热点
2. 电脑连接手机热点
3. 此时手机和电脑在同一网络
4. 使用局域网IP访问：`http://192.168.x.x:5173/`

---

### 方法3️⃣：使用内网穿透（临时）

#### 使用 ngrok（国外）

1. 下载 ngrok：https://ngrok.com/download
2. 运行：
```bash
ngrok http 5173
```
3. 会生成一个临时公网地址

#### 使用 cpolar（国内）

1. 下载 cpolar：https://www.cpolar.com/
2. 注册账号
3. 运行：
```bash
cpolar http 5173
```
4. 获得临时公网地址

---

## 📦 已构建文件

项目已构建到 `dist/` 目录，可以：
- 上传到任何静态网站托管服务
- 部署到自己的服务器
- 使用Nginx/Apache托管

| 方案 | 速度 | 费用 | 稳定性 | 难度 |

| Vercel | ⭐⭐⭐⭐⭐ | 免费 | 永久 | 简单 |

## ✅ 最佳建议

**如果是长期使用：** 部署到 Vercel 或 Netlify

**如果只是测试：** 手机开热点或使用内网穿透

---

## 📱 部署后如何使用

1. 获得公网地址后（如 `https://your-app.vercel.app`）
2. 在该网址上生成二维码
3. 手机扫码即可访问
4. 手机和电脑不需要在同一网络！
