# 快速排查"文件未找到"问题

## 立即检查（按顺序执行）

### 1. 检查 API 服务器是否运行

```bash
pm2 list
```

**应该看到：**
```
┌─────┬──────────────┬─────────┬─────────┬──────────┐
│ id  │ name         │ status  │ restart │ uptime   │
├─────┼──────────────┼─────────┼─────────┼──────────┤
│ 0   │ qrcode-api   │ online  │ 0       │ 5m       │
└─────┴──────────────┴─────────┴─────────┴──────────┘
```

**如果没有运行：**
```bash
cd /www/wwwroot/qrcode/2weima/api-server
pm2 start server.js --name qrcode-api
pm2 save
```

### 2. 测试 API 服务器

```bash
# 获取服务器IP
SERVER_IP=$(curl -s ifconfig.me)
echo "服务器IP: $SERVER_IP"

# 测试API
curl http://$SERVER_IP:3000/health
```

**应该返回：**
```json
{"status":"ok","timestamp":1234567890}
```

**如果返回错误：**
- 检查端口是否开放：`sudo ufw allow 3000`
- 检查云服务器安全组是否开放3000端口

### 3. 检查环境变量

```bash
cd /www/wwwroot/qrcode/2weima
cat .env.production
```

**应该看到：**
```env
VITE_API_BASE_URL=http://你的服务器IP:3000
```

**如果没有或不对：**
```bash
# 获取服务器IP
SERVER_IP=$(curl -s ifconfig.me)

# 创建/编辑环境变量文件
echo "VITE_API_BASE_URL=http://$SERVER_IP:3000" > .env.production

# 确认
cat .env.production
```

### 4. 重新构建项目

```bash
cd /www/wwwroot/qrcode/2weima
npm run build
```

### 5. 检查数据目录

```bash
# 查看是否有数据文件
ls -la /www/wwwroot/qrcode/2weima/api-server/data/

# 如果目录不存在，创建它
mkdir -p /www/wwwroot/qrcode/2weima/api-server/data
```

### 6. 查看 API 服务器日志

```bash
pm2 logs qrcode-api --lines 20
```

**应该看到请求记录：**
```
✓ 保存内容: file_1234567890
✓ 获取内容: file_1234567890
```

## 完整部署命令（一键执行）

```bash
# 1. 进入项目目录
cd /www/wwwroot/qrcode/2weima

# 2. 获取服务器IP
SERVER_IP=$(curl -s ifconfig.me)
echo "服务器IP: $SERVER_IP"

# 3. 配置环境变量
echo "VITE_API_BASE_URL=http://$SERVER_IP:3000" > .env.production
cat .env.production

# 4. 进入API服务器目录
cd api-server

# 5. 安装依赖（如果还没安装）
npm install

# 6. 启动API服务器
pm2 start server.js --name qrcode-api
pm2 save

# 7. 开放端口
sudo ufw allow 3000
sudo ufw reload

# 8. 测试API
curl http://$SERVER_IP:3000/health

# 9. 返回项目目录，重新构建
cd ..
npm run build

# 10. 重启Nginx
sudo systemctl restart nginx

# 11. 查看API服务器状态
pm2 list
pm2 logs qrcode-api --lines 10
```

## 测试步骤

1. **访问网站**：`http://你的服务器IP`
2. **生成一个文件二维码**（上传一个小文件测试）
3. **查看API日志**：
   ```bash
   pm2 logs qrcode-api --lines 5
   ```
   应该看到：`✓ 保存内容: file_xxx`
4. **用手机扫描二维码**
5. **再次查看API日志**：
   ```bash
   pm2 logs qrcode-api --lines 5
   ```
   应该看到：`✓ 获取内容: file_xxx`
